## Features

## Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript
- XAMPP

## Setup Instructions
1. Download or clone the repository
2. Import the database using phpMyAdmin
3. Rename db.example.php to db.php and update database name
4. Start Apache and MySQL using XAMPP
5. Open the project in the browser using localhost
